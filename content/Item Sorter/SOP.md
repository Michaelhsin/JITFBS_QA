# ITEM SORTER 教學 SOP

> 適用對象：供包台 / Presort 作業人員  
> 目的：確保供包與分揀流程正確，降低異常與誤刷風險

<br>

## 一、格口滑道正確 SOP
```flow
綁箱 → 開始落貨 → 滿箱 → 下箱
```
<br>

### 作業流程總覽
---
### 1️⃣ 綁箱

**操作步驟**
- 使用 PDA：
  1. 刷工位
  2. 刷 Device ID
- 系統完成綁定後即可開始作業

**燈號狀態**
- ⚪ 白色 → 🟡 黃色

**注意事項**
- 若未完成綁箱，貨是不會落於該格口中

### 📹 綁箱流程示範影片

<video src="/assets/images/bind_box.mp4" controls playsinline></video>

---

### 2️⃣ 開始落貨

**操作說明**
- 可安排人工協助，將箱內商品撥平，避免堆疊異常

![商品堆疊](/assets/images/stacked.jpg)

**燈號狀態**
- 🟡 黃色 → 🔵 藍色

![燈號代表意思](/assets/images/light_signal.jpg)


---

### 3️⃣ 滿箱

**判定條件**
- 商品遮蓋感應器連續 **6 秒**
- 系統自動停止落貨至該滿箱格口

**燈號狀態**
- 🔵 藍色 → 🔴 紅色

**重點**
- 滿箱為系統自動判定，禁止人為干預

![滿箱燈號示意](/assets/images/full.jpg)

---

### 4️⃣ 下箱

**操作步驟**
1. PDA 點選左上角 ☰ 選單
2. 選擇【解綁】
3. 刷工位
4. 確認解綁完成後下箱

**燈號狀態**
- 【一般下箱】 🔵 藍色 → ⚪ 白色 
- 【滿箱下箱】 🔴 紅色 → ⚪ 白色

**確認事項**
- 燈號回到白色才代表可進行下一箱

## 二、供包台作業流程（ITEM SORTER）
```flow
商品外包裝判斷 > 刷槍刷TO > 靜態掃刷SKU ID > 放貨
```
<br>

### 作業流程總覽


---

### 1️⃣ 商品外包裝判斷

#### 判斷項目

**① 明顯異常品**
- 超材
- 濕損
- 禁運品

**處理方式**
- 不可刷碼
- 直接放回原 Seller TO 箱
- 後續流程：
  - 專門驗收異常 Presort 播種牆
  - Presort 完畢後放異常箱
  - 交由 LRT 處理

**② 無標籤商品**
- 直接交給倉庫控台

---

### 2️⃣ 刷槍刷 TO（外箱）

**系統校驗規則**
- 可刷：
  - `1_xxx`
  - `2_xxx`
  - `SPTO`

**常見情境**
- 誤刷 SKU ID → 系統跳錯誤提示
- 外箱 TO 遺失：
  - 才可使用萬用 TO 或 SPTO

**提醒**
- 未確認 TO 前不可刷 SKU


---

### 3️⃣ 靜態掃刷 SKU ID

**系統校驗規則**
- 正確格式：`xxx_xxx`

**常見誤刷情境**
- 誤刷 Seller TO：
  - 系統仍可刷（Seller TO 為 `1_xxx`）
  - ⚠️ 後續會請廠商修正
- 誤刷賣家自製 UPC：
  - 系統跳錯誤訊息 **3 秒**
  - ⚠️ 會建議調整為 **1 秒**

### 4️⃣ 放貨（依供包台類型）


#### 🔹 自動供包台

**正確放貨方式**
- 商品需放在兩條紅線中間
  - 商品長度超過紅線距離時
  - 只要 **50% 商品位於紅線內** 即可感應
![商品放置示意](/assets/images/item_red_outline.jpg)
<br>

- 圓滾件：
  - 直放於皮帶上兩條溝槽中
![商品放置示意](/assets/images/item_round.jpg)
<br>

**小件商品異常處理**
- 若量測不到材積：
  - 商品停在皮帶上不前進
  - 螢幕材積欄位為空白

- 處理方式
    1. 嘗試將商品放在溝槽上重新量測
    2. 若仍失敗：
        - 螢幕點選【取消】 (⚠️務必先點選取消，否則會量測出錯誤的材積)
        - 將商品取下
        - 放入待 Presort 箱
        - 交由人工處理
        <br>

        <video src="/assets/images/no_measured.mp4" controls playsinline></video>

## 三、注意事項總結（必讀）

- 未刷 TO 不可刷 SKU
- 異常品第一時間不刷碼
- 滿箱為系統判定，不可硬塞
- 放貨位置錯誤將導致判讀異常
