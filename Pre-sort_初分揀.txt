【Pre-sort 初分揀】

---

## 流程
刷區域 QR code → 刷 ASN / Seller TO / SPTO → 檢查商品外觀 → 刷商品 UPC / SKUID → PDA 顯示工位 → 播種牆亮燈 → 投入揀貨箱 → 拍燈 → 綁箱（步驟1. 刷工位 QRcode → 步驟2. 刷 Device ID）→ 綁箱成功。

---

## 常見錯誤與處理方式

[錯誤1]
PDA錯誤訊息／異常狀況：此區域已超過三人
可能原因：區域人員超過三人
處理方式：重新選擇區域或等待空位

[錯誤2]
PDA錯誤訊息／異常狀況：無區域資料
可能原因：無區域資料
處理方式：在 Seatalk live issue 回報 @FBS PM 協助

[錯誤3]
PDA錯誤訊息／異常狀況：container_no 尚無資料
可能原因：SPTO 已無未分揀訂單
處理方式：請現場回報給 Joanna、Ashley、Toby

[錯誤4]
PDA錯誤訊息／異常狀況：訂單已無需預分揀資料，請刷讀其他訂單
可能原因：當刷 ASN / Seller TO 內已無未分揀訂單（含 JITSCS 整箱過期）
處理方式：交由 LRT（原因：整箱過期，因此不拆箱）

[錯誤5]
PDA錯誤訊息／異常狀況：sorting_group_shop_id: FBS 尚無資料，請群組回報
可能原因：系統內無該店家資料
處理方式：在 Seatalk live issue 回報 @vendor 協助

[錯誤6]
PDA錯誤訊息／異常狀況：條碼格式錯誤，請重新刷讀
可能原因：刷到非 ASN／Seller TO／SPTO 條碼
處理方式：重新刷讀正確條碼

[錯誤7]
PDA錯誤訊息／異常狀況：不支援南/北部訂單分揀，請整箱交總控
可能原因：整箱錯倉（因 WH 轉運所導致）
處理方式：交倉庫總控；總控開「錯倉表單」逐一刷 SKU 後整箱轉正確倉（見 SOP-01）

[錯誤8]
PDA錯誤訊息／異常狀況：人員區域未註冊，請先掃區域 QR code
可能原因：未選區域就刷 Seller TO 單
處理方式：先重刷區域 QR code

[錯誤9]
PDA錯誤訊息／異常狀況：如果 PDA 無訊息
可能原因：壞損品／超材／無條碼或條碼破損
處理方式：依情形放入 presort(有匹配/無匹配) 異常箱並交 LRT（見 SOP-02、SOP-03）

[錯誤10]
PDA錯誤訊息／異常狀況：訂單被取消／多貨（廠商多送）／該商品非屬於 Seller TO 訂單內
可能原因：條碼無對應訂單
處理方式：放入 presort(無匹配) 異常箱 → 退貨組（見 SOP-04）

[錯誤11]
PDA錯誤訊息／異常狀況：貨物退還給賣家門市，但賣家未取件
可能原因：此為 C2C 逆物流，該商品會退回倉庫等待報廢
處理方式：商品放入 presort(無匹配) 異常箱 → 退貨組

[錯誤12]
PDA錯誤訊息／異常狀況：EM 工位不存在
可能原因：材積攔截未設定 EM 工位
處理方式：Seatalk live issue 回報 @FBS PM

[錯誤13]
PDA錯誤訊息／異常狀況：播種牆卡燈（恆亮）／未拍燈
可能原因：前一次播種尚未拍燈
處理方式：先拍燈；必要時依 SOP-05 手動滅燈

[錯誤14]
PDA錯誤訊息／異常狀況：PDA 顯示工位後，播種牆未亮燈
可能原因：播種牆異常
處理方式：請依 SOP-06 操作

[錯誤15]
PDA錯誤訊息／異常狀況：因區域解除占用或未選區域，請重新選擇區域
可能原因：未選區域直接刷商品
處理方式：重新選區域後再作業

[錯誤16]
PDA錯誤訊息／異常狀況：此揀貨箱號尚未播種
可能原因：關箱重複（箱內已有 mainsort 資料）
處理方式：直接帶去分貨刷讀

[錯誤17]
PDA錯誤訊息／異常狀況：該工位商品已關箱完畢
可能原因：綁到空箱（無 mainsort 資料）
處理方式：請先播種新商品後再進行關箱

## SOP 細節

SOP-01：錯倉處理（南/北部不支援分揀）
說明：
1) 交由倉庫總控處理  
2) 總控開「錯倉表單」，逐一刷 SKU  
3) 整箱轉至正確倉。

SOP-02：壞損/超材（X 無訊息 - 情境 A）
說明：
- 發現當下不要刷，商品直接放回原 Seller TO 箱。  
- 之後在壞損/超材專用的 presort 播種牆處理。  
- 完成後放入 presort(有匹配) 異常箱 → 交 LRT。

SOP-03：無條碼／條碼破損（X 無訊息 - 情境 B）
說明：
- 請倉庫總控填寫 Google 表單後處理。  
- 商品放入 presort(有匹配) 異常箱 → 交 LRT。  
- 表單連結：https://forms.gle/p4RuZu4Q9M88T5aT7。

SOP-04：訂單取消／多貨／非該 Seller TO 訂單
說明：
- 商品放入 presort(無匹配) 異常箱 → 交退貨組。

SOP-05：播種牆卡燈／未拍燈
說明：
1) 檢查現場是否有未拍燈。  
2) 手動滅燈：登入 Web → 控制設定 → 手動解除滅燈。  
3) 若仍無法：在 Seatalk live issue 回報 @vendor 協助。  
4) 若頻繁發生：紀錄工位 → Seatalk live issue 回報 @vendor。

SOP-06：PDA 顯示工位後，播種牆未亮燈
說明：
1) 現場重啟控制器。  
2) 若無法處理，在 Seatalk live issue 回報 @vendor 協助。  
3) 若頻繁發生：紀錄工位 → Seatalk live issue 回報 @vendor。
